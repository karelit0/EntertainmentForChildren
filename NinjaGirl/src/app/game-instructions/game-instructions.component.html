NINJA GIRL

This is a simple game. Source code can be found on git

How to play
  • left arrow => move left
  • right arrow => move right
  • up arrow => jump
  • down arrow => move down
  • space bar => throw kunai
How to win
  • Do not touch fly enemies or the ninja girl is going to die.
  • Get the heart.
    • if one enemy gets the heart, he is going to dead but the max enemies is going to increase.
    • if the player gets the heart, the max enemies is going to decrease.
  • After one fly enemy die, there is a possibility that the max enemies is going to increase.
